## Installation

Pretty straightforward! Add the dependency to your `mix.exs`:
```
  def deps do
    [{:what3words, "~> 1.0.0"}]
  end
```

and add your API key to your `config.exs` (get one [here](https://map.what3words.com/register?dev=true)):
```
config :what3words, key: "mykey"
```
